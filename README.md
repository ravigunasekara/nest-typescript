## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ brew install node
$ npm i -g @nestjs/cli
$ nest --version
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Helpful Commands
```bash
$ nest new your-project-name
$ nest g module your-module-name
$ nest g controller your-controller-name --no-spec
$ nest g service your-service-name --no-spec
$ nest g pipe your-pipe-name --no-spec #Pipes are for custom validations
```
